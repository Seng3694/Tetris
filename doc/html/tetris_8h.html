void Tetris_LockInterrupt ( Tetris *  tetris)

Interrupts the current lock timer. Lock timer starts when a block is touching the floor and can't move down anymore. Everytime you move or rotate the block, the lock timer resets. The interruption is capped at LOCK_INTERRUPT_LIMIT. If the block won't get interrupted for LOCK_DELAY it will be solidified.

tetrisThe tetris game to interrupt.
